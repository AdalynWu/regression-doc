# Landing Page

::: info QA 團隊與新人 Onboard 知識點
**模組業務目的：**
Landing Page 是非海外地區全新使用者進站的第一門面。測試重點在於確保品牌視覺、導流按鈕、客服管道以及行動裝置適配性（RWD）維持正常，避免因排版錯亂或按鈕失效導致潛在用戶流失。

**環境與注意事項：**
1. **裝置驗證：** 必須包含桌面端（Desktop）以及行動端（Mobile/RWD）的網頁排版檢查。
2. **狀態限制：** 此頁面主要供「未登入」之訪客瀏覽，需驗證登入後的帳號不會被錯誤導回此頁。
:::
